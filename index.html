Project 2 • Climate Action Slide Presentation

Description

This project supports SDG 13: Climate Action. I built an interactive Python slide presentation showing the impact of rising temperatures on the environment and wildlife. Each slide reveals a new aspect—from CO₂ emissions to habitat destruction, ending with actionable solutions. It’s a powerful combination of visual storytelling and programming!

Concepts Learned

  • ✅ Slide-based storytelling with conditional logic
  • ✅ Custom Python functions and interactivity
  • ✅ SDG data visualization and awareness

Code Sample

# Climate Action Project – by Karl
def co2(x,y,color):
    Rect(x-30,y-30,60,60,fill=color)
    Line(x,y,x-20,y-20,fill='gold',lineWidth=3)    
    Line(x,y,x+20,y-20,fill='gold',lineWidth=3)
    Circle(x+20,y-20,7,fill='red')
    Circle(x-20,y-20,7,fill='red')
    Circle(x,y,10,fill='black')
    Label('CO2',x,y+25)

def Sun(x,y,color):

How It Solves the Goal

The slides raise awareness about global warming and its dangers—highlighting rising CO₂, wildlife threats, and climate-related statistics. It ends by showing solutions, like renewable energy and cooperation between countries. It’s a meaningful way to use code to educate and inspire climate action.
